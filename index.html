how to use a hexo theme

Hexo是一个快速、简洁高效的博客框架。在提供了多种可开发的博客主题的同时，hexo使用markdown语言解析文章，渲染生成静态网页（html文件）。

使用前准备：

  • Node.js (Node.js 版本需不低于 10.13，建议使用 Node.js 12.0 及以上版本)
  • 建立Repository
  • Git
  • 安装hexo
  • 在本地新建blog文件夹
  • 初始化blog文件夹 对blog文件夹右键或打开后在空白背景上右键，使用git bash here

参考视频：
[一步一步教你如何在Windows10上完成hexo博客搭建和yilia主题使用]（https://www.bilibili.com/video/BV1pt411K7o9?p=2）

将hexo关联到github page上

由于本人使用的是私有repository，并不希望公开站点文件，因此不使用Travis CI进行部署。

  1. 下载hexo deploy插件
  2. 在blog所在文件夹中找到_config.yml配置文件，修改deploy项

    deploy:
    type:git
    repo:<仓库url地址>如：https://bitbucket.org/JohnSmith/johnsmith.bitbucket.io
    branch：分支名称
    message：修改信息
